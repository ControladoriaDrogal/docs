# Documentações - Controladoria Drogal

Este repositório _Git_ centraliza as documentações técnicas e regras de negócio desenvolvidas pela **Controladoria**.

## Objetivo

Toda documentação funcional/técnicas, critérios, regras, memórias de cálculo, visões SQL deverão ser geridas por este projeto, garantindo:

- Rastreabilidade das premissas
- Padronização documental
- Apoio à manutenção dos processos da controladoria
- Fonte única de referência para regras e critérios
